Frequently Asked Question

Q. What is this web application?

        A. This web application allows to check which devices are logging to Splunk. Simply place a hostname in to check if a device is logging

 

Q. What is the difference between the two forms?

        A. The first form check if a device is logging to splunk.
             The second form checks if a Universal Forwarder is connecting to our infrastructure

 

Q. How up to date is the data?

        A. The data is in real-time
